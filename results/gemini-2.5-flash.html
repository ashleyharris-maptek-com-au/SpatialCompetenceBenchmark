Typical Prompt:
You have 3 1m lengths of pipe, and a square area of side length 1 to play with.

Lay the pipe out to form a closed loop, using all the pipe, and returning to the starting point.

You can not re-use vertices, and you can not cross the boundary of the area. You do not need to
stick to axis aligned paths.

Return the loop as a list of the pipe endpoints. Note that N pipes requires N+1 verticies to describe a path, but since the
first and last vertices are the same, you only need to return N points.
The loop must not cross itself or touch itself.
 Prompt Changes:
Increasing pipe length, square size, and complications.

  • 21-30 require some unusual shapes, as the solution typically requires concentric circles or railway-track-like geometries
  • 31-40 require either clever algorithms, or special solvers run using code execution. The human-with-tools solver uses a spring simulation written in python.
