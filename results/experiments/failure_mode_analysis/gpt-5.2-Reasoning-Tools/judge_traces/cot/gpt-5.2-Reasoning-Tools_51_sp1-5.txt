**Classifying failure modes**

I'm analyzing the failure modes here. It seems that for subpasses 1-4, empty configurations are being outputted when the ground truth is non-empty. There are many components missing, indicating a local-only or global integration failure. It’s not about evasion; the format is valid. Overall, the main error type is simply "missing." For subpass 5, there's an extra configuration that turns out to be a near-miss edge case. So I see a pattern of missing and an extra one being a near-miss.
