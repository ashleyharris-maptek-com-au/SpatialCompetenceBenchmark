Here's the OpenSCAD file to create the shape of the sand pile:

```
// Sand pile in a silo

// Constants
angle_of_repose = 33;
silo_diameter = 10 * 0.9144; // Convert yards to meters

// Volume conversions
cubic_yards_to_meters = 5 * 0.764555; // Convert cubic yards to cubic meters
cubic_feet_to_meters = 1500 * 0.0283168; // Convert cubic feet to cubic meters
train_volume = 100; // Already in cubic meters
drum_volume = 20 * 44 * 0.00454609; // Convert UK Imperial Gallons to cubic meters

// Total volume
total_volume = cubic_yards_to_meters + cubic_feet_to_meters + train_volume + drum_volume;

// Height of the cone part of the sand pile
cone_height = total_volume / (pow(tan(angle_of_repose * PI / 180), 2) * PI * pow(silo_diameter / 2, 2));

// Cylinder height after accounting for the cone part
cylinder_height = cone_height * (1 - cos(angle_of_repose * PI / 180));

// Create the sand pile shape
difference() {
    translate([0, 0, cylinder_height])
    cylinder(h = cone_height, d = silo_diameter, $fn = 100);
    cylinder(h = cylinder_height, d = silo_diameter, $fn = 100);
}
```

This OpenSCAD file calculates the volume of sand from each source, converts the volumes to cubic meters, and calculates the height of the cone and cylinder parts of the sand pile based on the angle of repose. The final shape is created using the `difference()` function to subtract the lower cylinder from the combined cylinder and cone shape.
