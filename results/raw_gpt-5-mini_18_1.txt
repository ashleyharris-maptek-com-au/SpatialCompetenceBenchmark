// OpenSCAD file: Tallest pyramid with 150,000 stones (each 0.500 m cube)
// Units: meters (m)
// Each layer is represented as a solid slab composed of k x k cubes (merged into one cuboid).
// This preserves exact overall volume and outer geometry while remaining CAD-friendly.
//
// Summary of computation:
// - Cube side (stone size) s = 0.500 m
// - Maximum full pyramid height (number of layers) is the largest n with sum_{k=1..n} k^2 <= 150000
//   That n = 76 (sum_{k=1..76} k^2 = 149226 stones).
// - Base layer = 76 x 76 stones -> base footprint = 76 * 0.5 m = 38.000 m per side.
// - Total stones used = 149226 (149,226 stones). Total solid volume = 149226 * (0.5^3) m^3.
// - Total pyramid height = 76 * 0.5 m = 38.000 m.
// - Pyramid is centered at the origin (0,0) and faces north (+Y). Layers are axis-aligned.
//
// The model builds 76 slab layers (k x k x s) stacked with their centers aligned at X=0, Y=0.
// The bottom of the pyramid sits on Z=0; the center of layer i (0-based from bottom) is at
//   z = (i + 0.5) * s.
//
// This representation gives millimeter-level positional precision (3 decimal places) and
// is optimized by merging all stones within a layer into a single cuboid primitive,
// yielding a compact, viewable model suitable for stonemason CAD work.

s = 0.500;              // stone side length (m)
max_layers = 76;        // computed maximum layers given stone budget
total_stones_used = 149226; // sum_{k=1..76} k^2
base_side = max_layers * s; // 38.000 m
total_height = max_layers * s; // 38.000 m

// optional: color to visually indicate the pyramid (remove if not wanted)
module pyramid()
{
    // Build from bottom layer (level=0) up to level=max_layers-1
    union()
    {
        for (level = [0 : max_layers - 1])
        {
            k = max_layers - level;                // number of stones on this layer's side
            size = k * s;                          // full slab side length (m)
            zcenter = (level + 0.5) * s;          // center height of this slab (m)
            // Place the slab centered in X and Y, with its center at zcenter.
            translate([0, 0, zcenter])
                cube([size, size, s], center = true);
        }
    }
}

// Call the module to render the pyramid
pyramid();
