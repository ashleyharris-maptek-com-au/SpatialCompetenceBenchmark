I need to compute the Delaunay triangulation for the given set of 2D points.

Let me list the points:
- Point 0: [0.956, 0.208]
- Point 1: [0.828, 0.149]
- Point 2: [0.513, 0.136]
- Point 3: [0.689, 0.842]
- Point 4: [0.426, 0.957]
- Point 5: [0.825, 0.338]

Delaunay triangulation is a triangulation where no point is inside the circumcircle of any triangle. I can compute this using scipy's Delaunay function.

Let me write a Python script to compute this:

```python
import numpy as np
from scipy.spatial import Delaunay

points = np.array([
  [0.956, 0.208],
  [0.828, 0.149],
  [0.513, 0.136],
  [0.689, 0.842],
  [0.426, 0.957],
  [0.825, 0.338]
])

tri = Delaunay(points)
triangles = tri.simplices

# Sort each triangle in ascending order
triangles_sorted = [sorted(triangle.tolist()) for triangle in triangles]

print(triangles_sorted)
```
